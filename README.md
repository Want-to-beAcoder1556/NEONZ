# NEONZ
A zombie defender game where you can get more gun barrels, different bullets, drones to auto shoot, a shield to protect yourself, and upgrade your health to last longer when zombies are hurting you. Every wave, the zombies get a bit more health to counter your upgrades...            stay strong and get as far as you can! 
There are so many bullets!
1. Regular bullet. This bullet cannot be bought as you start with it; there are no upgrades.
2. Tracker bullet, this bullet has 3 upgrade stages: 1 slow targeting, 2 mild targeting, 3 fast targeting.
3. Slap bullet, this bullet acts like the regular bullet but targets any zombies that come within a close proximity (distance) of the player.
4. Breaking Bullet, this bullet acts like the regular bullet, but when it gets far from the player, it breaks into three smaller bullets that Target zombies; these smaller bullets do less damage.
5. Piercing bullet, this bullet has 10 upgrades, and each upgrade allows the bullet to go through more zombies before disappearing.

There are four upgrades to the drones, and each upgrade gives one more drone to auto shoot zombies that are near you.

The shield has three upgrades, each of which improves the durability of the shield. If no zombies are attacking the shield, it will regenerate durability.
